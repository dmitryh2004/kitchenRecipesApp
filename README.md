# Приложение рецептов

Предназначено для возможности хранить свои рецептики в одном месте, а также делиться ими с другими пользователями.

## Возможности

- Специально для тех, кому надоел вопрос "Что на ужин?": возможность сохранить варианты ужина и при следующем таком вопросе воспользоваться приложением в качестве подсказки;

Остальное в данный момент находится в разработке.

## Известные проблемы

- Антивирусы на платформе Аваст могут распознавать приложение как вирус evo-gen: эти предупреждения можно игнорировать, так как они являются ложно-положительными.
